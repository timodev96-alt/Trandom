# Want your own version?
### 1. First Download the Dependencies
1. Kotlin Standard Library
2. Android SDK: Platform 33 (android.jar).
3. JDK: Java JDK 21.
4. Build Tools: Android Build Tools 33.0.2.

### 2. Change `local.properties` file to your file paths 
```
sdk.dir= "\AndroidSDK"
jdk.bin= "\Java JDK 21\bin"
kotlin.lib= "\Kotlin\kotlinc\lib"
```
### 3. Simpley run `run.bat`
#### Make sure you have your phone connected & USB DEBUGGING is Enabled... also Set your phone to "Transfer media" mode

### Or Just Download the apk [here](https://github.com/timodev96-alt/Trandom/releases/download/Main/Trandom.apk)!
